Synclick - Multi-Device Clock Synchronization Program

Overview

Synclick is a multi-device clock synchronization program developed by the Human Factors Engineering Research Group at Shenzhen University. Built with Qt C++, it enables millisecond-level synchronous data collection across multiple devices, providing high-precision timing control solutions for scientific research and engineering applications.

Key Features

  • Millisecond-level synchronization precision
  • Supports multi-device collaborative work
  • Real-time status monitoring and feedback
  • Multiple synchronization modes and configurations
  • User-friendly interface built with Qt C++
